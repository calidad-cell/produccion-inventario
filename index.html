🐟 Sistema de Inventario

Gestión de producción, mermas y stock para proteínas procesadas. Conecta con Google Sheets para guardar en la nube.

  1. Crea una planilla en Google Sheets
  2. Ve a Extensiones → Apps Script
  3. Pega el código del comentario al final de este HTML
  4. Despliega como webapp y copia la URL
💡 Los productos de tu Excel ya están precargados. Puedes usar el modo demo sin configurar nada.
